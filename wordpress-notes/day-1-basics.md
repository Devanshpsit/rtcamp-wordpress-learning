## What is WordPress?
WordPress is an open-source Content Management System (CMS) built using PHP and MySQL.

## WordPress.com vs WordPress.org
WordPress.org gives full control over themes, plugins, and code.
It is used for professional and enterprise-level development.

## Core WordPress Structure
- wp-admin
- wp-content
- wp-includes

The wp-content folder contains themes, plugins, and uploads.

## Themes
Themes control the design and layout of a WordPress website.

## Plugins
Plugins add extra functionality to WordPress websites.

## Hooks
Hooks allow developers to add or modify WordPress functionality without editing core files.
